Ромб и вписанная окружность
Геометрия
Острый угол ромба равен 30°. Радиус окружности, вписанной в этот ромб, равен 2. Найдите сторону ромба.

Для ромба со стороной a и острым углом α площадь равна a² · sin α. Полупериметр равен 2a. Радиус вписанной окружности:

r = S / (полупериметр) = a² · sin α / (2a) = a · sin α / 2

Откуда:

a = 2r / sin α = 2 · 2 / sin 30° = 4 / 0,5 = 8
Ответ: 8

Перелив жидкости между цилиндрами
Геометрия
В цилиндрическом сосуде уровень жидкости достигает 16 см. На какой высоте будет находиться уровень жидкости, если её перелить во второй цилиндрический сосуд, диаметр которого в 2 раза больше диаметра первого? Ответ дайте в сантиметрах.

Объём жидкости постоянен. Пусть радиус первого сосуда — r. Тогда радиус второго — 2r.

π · r² · 16 = π · (2r)² · h₂ r² · 16 = 4r² · h₂ h₂ = 16 / 4 = 4 см
Ответ: 4

N = 6! / (2³ · 3!) = 720 / (8 · 6) = 15

Линейная комбинация при заданном отношении
Алгебра
Найдите 61a − 11b + 50, если (2a − 7b + 5) / (7a − 2b + 5) = 9.

Из данного условия: 2a − 7b + 5 = 9 · (7a − 2b + 5) = 63a − 18b + 45.

Отсюда: 2a − 63a − 7b + 18b + 5 − 45 = 0  →  −61a + 11b − 40 = 0.

61a − 11b = −40 61a − 11b + 50 = −40 + 50 = 10

Универсальный метод. Представим 61a − 11b + 40 как линейную комбинацию числителя N = 2a−7b+5 и знаменателя D = 7a−2b+5:

α·N + β·D = 61a − 11b + 40 2α + 7β = 61 ; −7α − 2β = −11 → α = −1, β = 9

Тогда 61a − 11b + 40 = −N + 9D. Подставляем N = 9D (из условия):

= −9D + 9D = 0 → 61a − 11b + 50 = 0 + 10 = 10
Ответ: 10

Прикладная математика

v = 0,5 · sin(2π · 4 / 24) = 0,5 · sin(π/3) = 0,5 · (√3/2) = √3/4 ≈ 0,433 м/с E = m · v² / 2 = 0,08 · (√3/4)² / 2 = 0,08 · (3/16) / 2 = 0,08 · 3/32 = 0,0075 Дж

Наименьшее значение тригонометрической функции
Тригонометрия / Производная
Найдите наименьшее значение функции вида y = A − Bπ + C·x − D·sin(x) на отрезке [0; π/2].

Функция имеет вид y = const + C·x − D·sin(x). Производная:

y′ = C − D·cos(x) = 0 → cos(x) = C/D

Для данного варианта критическая точка x₀ = π/4 (cos(π/4) = 1/√2). Константный член (содержащий π) точно отменяется линейным слагаемым C·(π/4) в точке минимума. После подстановки:

y(π/4) = [A − B·π + C·(π/4)] − D·sin(π/4) = A − D·(√2/2) = −34

Проверка краёв и сравнение подтверждают, что −34 — глобальный минимум на [0; π/2].

Ответ: −34
