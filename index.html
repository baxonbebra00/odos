r = S / (полупериметр) = a²·sin α / (2a) = a·sin α / 2

Алгебра

2a − 7b + 5 = 63a − 18b + 45 −61a + 11b − 40 = 0 61a − 11b = −40 61a − 11b + 50 = −40 + 50 = 10

Метод линейной комбинации: 61a−11b+40 = α·N + β·D, где N = 2a−7b+5, D = 7a−2b+5. Система: 2α+7β=61, −7α−2β=−11. Решение: α=−1, β=9. При N=9D: −N+9D = −9D+9D = 0, откуда 61a−11b+50 = 10.

Ответ: 10

Линейная комбинация при заданном отношении
Алгебра
Найдите a + b + 8, если (6a − 7b + 4) / (7a − 6b + 4) = 1.

При отношении = 1 числитель равен знаменателю:

6a − 7b + 4 = 7a − 6b + 4 6a − 7a − 7b + 6b = 0 −a − b = 0 → a + b = 0 a + b + 8 = 0 + 8 = 8
Ответ: 8

Кинетическая энергия груза на пружине
Прикладная математика
m = 0,8 кг, v₀ = 0,3 м/с, T = 24 с. Найдите E через 9 с.
v = 0,3·sin(2π·9/24) = 0,3·sin(3π/4) = 0,3·(√2/2) = 0,15√2 м/с E = 0,8·(0,15√2)² / 2 = 0,8·0,045 / 2 = 0,018 Дж

Примечание: sin(3π/4) = sin(π/4) = √2/2; v² = 0,09·(1/2) = 0,045.

Ответ: 0,018 Дж
